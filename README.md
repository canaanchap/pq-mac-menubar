### Progress Quest - macOS Menubar Application

"Progress Quest is an antic and fantastical computer role-playing game, wellworth the time you'll spend playing it. And it won't cost you a farthing!"

A port (plus more) of 2002's Progress Quest. I want to "play" on my Mac, so here we are!

Inspired by: http://progressquest.com/
Started from: https://bitbucket.org/grumdrig/pq

Completely vibe-coded except for some of the data-driven elements, which have been edited and expanded.

*Note to me:
To run a dev build:
`./scripts/dev-build.sh`
`swift run PQMenuBarApp`

Versions:
- 8.x.x: the multiplayer additions
